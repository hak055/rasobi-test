1. Верстка.
   - Шаблон для верстки https://www.figma.com/file/HbqLlG4n1FfNFkWVoaFicZ55/Test?node-id=0%3A1 
          (если зарегистрироваться, то появится возможность смотреть размеры, цвета и т.д.)
   - В мобильной версии при клике по гамбургеру (в правом верхнем углу) должен выезжать сайдбар с простым меню.
   - В мобильной версии в блоке "Наша компания в цифрах" можно выстраивать эти самые цифры в столбик
          (можно сделать слайдером как бонус, но не обязательно).
   - В мобильной версии блоки меню в футере можно тоже сделать один под другим в столбик
          (без всяких аккордионов).
   - Желательно, но не обязательно для верстки использовать какой либо шаблон для верстки с Gulp, Sass (Scss).
     Можно использовать bootstrap, можно Smartgrid (https://github.com/dmitry-lavrik/smart-grid). 
	 Либо что то другое, на свой выбор. Ограничений нет.

2. Прикрутить верстку к October CMS, используя плагин StaticPages.
   - сделать отдельную тему (и в ней же в какую нибудь папку, например "src", положить исходники с версткой)
   - В админке должна быть возможность поменять номера телефонов и значения в блоке 
         "Наша компания в цифрах" (сделать в настройках темы).
   - Сделать возможность добавлять вопросы с ответами и выводить их только в центральной части. 
   Боковю колонку можно не трогать.
     Добавление можно сделать просто в виде repeater-а прям в шаблоне.
	 При большом желании можно сделать в виде отдельного плагина с сохранением в базе и выводом в виде компонента.
   - Поиск делать не нужно.

3. Выложить все куда нибудь через git (github, bitbucket и т.п.). 
    Лучше не одним коммитом, а поэтапно с историей изменений.
